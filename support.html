Thank you for using RouteViews! This page provides helpful information about using the app, troubleshooting common issues, and getting further support.

About RouteViews

RouteViews is a mobile application that gives you access to global BGP routing data through the University of Oregon Route Views Project. The app allows you to connect to public RouteViews routers around the world and query real-time BGP routing information for any subnet.

Getting Started

  1. Select a Router: Choose from the list of available RouteViews routers from different continents.
  2. Enter a Subnet: Input the subnet you want to query in CIDR notation (e.g., 192.168.1.0/24).
  3. Run Query: Tap "Query BGP" to connect to the selected router and retrieve the AS path information.
  4. View Results: Examine the retrieved AS path data and raw router output.

Frequently Asked Questions

General Questions

  • What is BGP?

    BGP (Border Gateway Protocol) is the protocol that makes the internet work. It's responsible for routing traffic between different networks (Autonomous Systems) across the internet.

  • What is an AS path?

    An AS path is the sequence of Autonomous System numbers that a network prefix traverses to reach its destination. It shows the routing path through the internet.

Troubleshooting

  • Connection Failed

    If you're having trouble connecting to a router:

    • Check your internet connection
    • Try a different router
    • Your network may be blocking SSH connections (port 22)
    • The selected router may be temporarily unavailable
  • Invalid Subnet Format

    Ensure you're using proper CIDR notation (e.g., 8.8.8.0/24) when entering a subnet.

  • Slow Connection

    RouteViews routers are located around the world, and connection times can vary based on your location and the selected router. Try selecting a router geographically closer to your location for faster response times.

Features

  • Global Router Selection: Access routers in North America, Europe, Asia, Australia, and more.
  • Real-time BGP Data: Get current routing information directly from RouteViews routers.
  • Query History: Save and review past BGP lookups.
  • Custom Routers: Add your own RouteViews compatible routers.
  • Terminal Output: View raw router command output for detailed analysis.

Privacy & Data Usage

RouteViews only collects data necessary for the functioning of the app:

  • We do not collect personal data
  • Your queries are not stored on any external servers
  • Query history is stored locally on your device only
  • The app communicates directly with public RouteViews routers
